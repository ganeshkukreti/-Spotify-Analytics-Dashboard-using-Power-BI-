# 🎧 Spotify Music Analytics Dashboard (Power BI Project)

## 📌 Project Overview

This project is an interactive **Spotify Music Analytics Dashboard** built using **Power BI** to analyze song performance, artist trends, popularity metrics, and listening insights.
The dashboard helps users explore patterns in music popularity across artists, albums, and months using dynamic visualizations.

This project demonstrates my ability to perform **data cleaning, transformation, modeling, and dashboard storytelling** using Power BI.

---

## 🎯 Objectives

* Analyze song popularity trends
* Identify top-performing artists and tracks
* Compare explicit vs non-explicit content distribution
* Track monthly popularity variations
* Explore album-type level performance insights

---

## 📊 Dashboard Features

### 1️⃣ Overview Page

Includes:

* Total distinct songs
* Average popularity score
* Average song duration insights
* Explicit vs Non-explicit comparison
* Songs by album type distribution
* Monthly popularity trend visualization

### 2️⃣ Artist Analysis

* Top artists by song count
* Artist-level performance comparison
* Popular tracks contribution per artist

### 3️⃣ Song-Level Insights

* Top songs by popularity
* Song performance comparison
* Monthly distinct song trends
